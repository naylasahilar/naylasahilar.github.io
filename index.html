Pilih Bahasa Pemrograman

HTML & CSS

Bahasa digunakan untuk membuat
dan mendesain tampilan setiap
situs web.

Java Script

Bahasa fleksibel yang digunakan
di mana saja, mulai dari situs web
interaktif hingga server backend.

Node.js

Runtime Environment JavaScript
untuk mengembangkan aplikasi
server-side.

React

Library JavaScript front-end yang
banyak digunakan oleh
programmer global.

SQL

Bahasa database digunakan untuk
manipulasi dan analisis database.

Python

Sebuah bahasa pemrograman
populer yang di gunakan untuk data
science, machine learning, dan
banyak lagi.

Ruby

Bahasa yang dinamis, serba-guna,
biasanya digunakan untuk
membangun aplikasi-aplikasi
berbasis web.

Ruby In Rails5

Sebuah framework yang
mencakup semua yang dibutuhkan
untuk membuat aplikasi web.

Java

Bahasa populer yang bisa digunakan
untuk pengembangan perangkat
lunak, aplikasi seluler, dan lainnya.

PHP

Bahasa server-side yang cocok
digunakan untuk
pengembangan web.

Go

Sebuah bahasa pemrograman
yang diproduksi oleh Google.

Sass

Cara mudah menuliskan CSS.
